# ⏱️ ChronoX | Modern Responsive Stopwatch

A modern, responsive, and interactive stopwatch web application built using **HTML**, **CSS**, and **JavaScript**. ChronoX combines accurate time tracking with a clean glassmorphism-inspired interface, animated backgrounds, and a smooth user experience across desktop and mobile devices.

---

## 📌 About

ChronoX is a premium stopwatch application that demonstrates the use of JavaScript timing functions, DOM manipulation, responsive web design, and modern UI principles. The project provides users with an intuitive interface to accurately measure time while maintaining an attractive and responsive design.

---

## ✨ Features

- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Record Unlimited Lap Times
- 📋 Dynamic Lap History
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- 🌈 Animated Gradient Background
- ⚡ Smooth Hover Effects & Animations

---

## 🛠️ Technologies Used
